# Simple Movie Website

# How to clone repository

1. copy url "https://github.com/handikaa/Simple-movie-website-HTML-CSS.git"
2. open terminal/cmd on your device go to your folder project or folder to clone the project
3. type "git clone {paste url repository}" and type enter
4. open code edtior and open project folder in your code editor

# How to run the project

1. open folder project on your file.
2. find file name "index.html" and double tap
